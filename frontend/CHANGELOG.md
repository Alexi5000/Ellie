# Changelog

All notable changes to the Ellie Voice Receptionist frontend will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-13

### Added
- Initial release of Ellie Voice Receptionist frontend
- Marketing site with responsive design
- Voice interaction interface
- Real-time WebSocket communication
- Dark/light theme support
- Internationalization (i18n) support
- PWA capabilities
- Accessibility features (WCAG 2.1 AA compliant)
- Performance monitoring
- Cross-browser compatibility
- Comprehensive test suite

### Features
- **Marketing Site**: Complete marketing page with hero, features, solutions, and more
- **Voice Interface**: Real-time voice interaction with AI assistant
- **Theme System**: Dark/light mode with system preference detection
- **Responsive Design**: Mobile-first design that works on all devices
- **Accessibility**: Full keyboard navigation and screen reader support
- **Performance**: Lighthouse score 95+, optimized bundle size
- **Testing**: Unit, integration, and E2E tests with 80%+ coverage

### Technical Stack
- React 18.2
- TypeScript 5.3
- Vite 5.0
- Tailwind CSS 3.3
- Framer Motion 12.x
- Socket.io Client 4.7
- React Router 6.20
- i18next 25.x
